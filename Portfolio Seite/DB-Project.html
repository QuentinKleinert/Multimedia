Ontoexplorer

Software Project

The OntoExplorer provides visibility into Semantic Web SPARQL endpoints and enables exploration through, research for, and filtering of RDF datasets in these endpoints. To do this, the endpoints can be managed manually, the result list can be enriched with attributes and the page view and the configuration of the endpoints can be shared with other users.
